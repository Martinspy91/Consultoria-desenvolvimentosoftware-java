Sistema de Consultoria para Desenvolvimento de Software

📌 Sobre o projeto

Este projeto foi desenvolvido em grupo como trabalho acadêmico, utilizando Java no NetBeans. O objetivo foi simular o gerenciamento de uma empresa de consultoria em desenvolvimento de software, aplicando conceitos de Programação Orientada a Objetos (POO), interfaces gráficas e persistência de dados.

O sistema permite gerenciar clientes, colaboradores e projetos, além de realizar a alocação de colaboradores aos projetos. Os dados são persistidos, permanecendo salvos mesmo após o fechamento da aplicação.

🚀 Funcionalidades

- Login de usuário
- Menu principal
- Cadastro de clientes
- Cadastro de colaboradores
- Cadastro de projetos
- Alocação de colaboradores aos projetos
- Persistência de dados

🖥️ Telas do sistema

- Tela de Login
- Menu Principal
- Cadastro de Clientes
- Cadastro de Colaboradores
- Cadastro de Projetos
- Alocação de Colaboradores

🛠️ Tecnologias utilizadas

- Java
- NetBeans IDE
- Java Swing (Interface Gráfica)
- Persistência de dados em arquivos

📂 Estrutura do projeto

- Classes de Cliente
- Classes de Colaborador
- Classes de Projeto
- Classes de Alocação
- Interface gráfica (telas)
- Persistência de dados

🎯 Objetivo

Desenvolver uma aplicação desktop para praticar conceitos de Programação Orientada a Objetos, criação de interfaces gráficas, organização de código e persistência de dados em Java.

👥 Equipe

Projeto desenvolvido em grupo como atividade acadêmica da graduação em Sistemas de Informação.
